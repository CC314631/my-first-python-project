# 密码强度检测工具 (Password Strength Checker)

## 📋 项目介绍
一个基于 Python 编写的命令行密码安全检测工具。
通过分析密码长度、字符组成（大小写字母、数字、特殊符号），智能评估密码安全强度，并给出优化建议。

## 🌟 核心功能
- ✅ 长度检测：自动识别密码位数
- ✅ 字符分析：识别是否包含大写、小写、数字、特殊符号
- ✅ 强度评级：采用 100 分制，评级为 `强` / `中` / `弱`
- ✅ 智能建议：针对弱密码提供具体的优化方案
- 🚀 支持多次检测，输入 `exit` 即可退出程序

### 环境要求
- Python 3.6+

##运行程序
python PasswordChecker.py
